# 老師專用 PDF 轉圖片小工具

**完全在瀏覽器本地執行，不上傳任何檔案，隱私安全！**

適合把講義、試卷、教科書頁面、作業快速轉成 JPG / PNG 圖檔，方便插入 PowerPoint、列印註記、或做數位筆記。

### 🚀 立即使用（最推薦）
👉 **[直接開啟工具](https://prochemistry-lab.github.io/pdf-to-image-teacher-tool/)**

### 📥 下載離線版本（適合學校斷網使用）
稍後我會教你建立 Release，一鍵下載 ZIP

### 使用方法（超簡單）
1. 點上方「直接開啟工具」或下載後雙擊 `index.html`
2. 拖拉 PDF 檔案進去（或點擊上傳）
3. 選擇畫質（高解析 / 快速）
4. 點「轉換」→ 自動下載所有圖片（或整包 ZIP）

### 特色
- 純 HTML + JavaScript，**完全離線可用**
- 支援多頁 PDF 批量轉換
- 不需要安裝任何軟體
- 資料不會離開你的電腦
